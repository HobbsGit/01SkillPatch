Git init : create new project
Touch - creating a new file
Git Add : adding to staging (tracking on)
Git status : status of tracking
Git Commit: Commit to staging
Git Commit -m "<comments>": Commit to staging
Git rm -cached <filename> : remove from staging
Git remote add origin <repositorypath> : Add to remote repository
Git push -u origin master :  to push into remote repository
Git Clone <repositorypath>: load repository
Git Checkout <repositorypath>: switching between branches
git pull : pull latest from server 

In vim, 
i to start entering text and save by pressing esc 
:wq and enter - this will commit with the message you typed
:q and enter - just come out without committing
